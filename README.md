# Timer Assignment

## 구현 기능
- 초 단위로 증가하는 카운트업 타이머
- Start / Pause / Reset 동작 지원
- 목표 시간 설정 및 도달 시 알림 표시
- 현재 타이머 상태 표시 (Idle / Running / Paused)

## 설계 및 구현 방향
- 타이머의 동작을 Idle, Running, Paused 상태로 구분하여 관리했습니다.
  상태에 따라 버튼 활성 여부와 동작이 자연스럽게 결정되도록 구성했습니다.
- 시간 계산은 setInterval로 값을 누적하는 방식 대신,
  기준 시각과 현재 시각의 차이를 계산하는 방식을 사용했습니다.
  이를 통해 장시간 실행 시에도 시간 오차가 누적되지 않도록 했습니다.
- 목표 시간에 도달하면 타이머가 자동으로 정지되도록 하여,
  완료 시점을 명확히 인식할 수 있도록 했습니다.
